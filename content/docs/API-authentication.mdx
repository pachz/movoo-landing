kind: "simple"
parent: "API Docs"
---

**Header Format:**
```bash
{
Authorization: Bearer <your_api_key>
```

> Replace `<your_api_key>` with the actual key provided in your onboarding dashboard.

---

### 401 Unauthorized Response
```bash
{
  "success": false,
  "error": "unauthorized",
  "message": "Invalid/missing token"
}
```
